# Parlok Infra Project

## Description
Parlok Infra is an ASP.NET + SQL Server web application to manage construction projects, track progress, and store project data securely.

## Features
- Project Dashboard
- User Management
- Image and Document Upload
- Reports Generation

## How to Run
1. Open the solution in Visual Studio 2022 or later.
2. Restore NuGet packages.
3. Update the connection string in `appsettings.json` to point to your SQL Server.
4. Run the project.
